# 3-bit Synchronous Up/Down Counter Design

### Project Overview
This project demonstrates a high-precision **3-bit Synchronous Up/Down Counter** designed using **JK Flip-Flops**. It features a control signal to switch between counting sequences, ensuring high reliability for digital timing applications.

### Technical Specifications
* **Logic Framework:** Built using excitation tables and K-Maps for optimized Boolean expressions.
* **HDL Language:** Implemented in **VHDL** with a focus on modular code structure.
* **Simulation:** Verified using **EDA Playground** to ensure zero-error waveforms.
